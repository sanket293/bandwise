## Notes / how these were made
- Captured on an Android emulator (Pixel-class, 1080×2400) running the debug build.
- The sample practice entries and rising trend were entered by hand purely for
  the listing — they are not shipped with the app.
- Ads are hidden via the app's "Focus Mode" (watch-to-hide-banners) so no
  test-ad banner appears. The store screenshots are therefore ad-free.
